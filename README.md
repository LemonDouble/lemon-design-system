# LemonDouble Design System

AI 에이전트가 읽을 수 있는 개인 디자인 시스템. 프로젝트에 `DESIGN.md`를 복사하고 "이 디자인 시스템으로 만들어"라고 하면 일관된 LemonDouble 스타일이 적용됩니다.

## Files

| File | Description |
|------|-------------|
| `DESIGN.md` | 디자인 시스템 전체 스펙 (컬러, 타이포, 컴포넌트, 레이아웃 등) |
| `preview.html` | 컴포넌트 시각 카탈로그 (26개 컴포넌트) |
| `images/` | 로고, 아바타, 마스코트 원본 이미지 |

## Key Decisions

- **Colors**: Logo Yellow `#F0B90B` + Coral `#CD6B5E` on Warm Dark `#12100E`
- **Fonts**: Galmuri11 Bold (headings only) + Pretendard (everything else)
- **Icons**: Lucide (UI) + Simple Icons (brands)
- **Theme**: Dark mode only
